## Format

Each script includes:
- **Emotional beats** in brackets: `[sighs]`, `[excited]`, `[mischievously]`, etc.
- **Natural pauses** and conversational flow
- **CTA**: "Link below" ending

## Usage

These scripts are designed for:
- UGC-style video ads (Arcads, Captions, etc.)
- Testimonial-format content
- Social media ads (Meta, TikTok, Reddit)
